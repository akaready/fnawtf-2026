## HTML

```html
<div class="link-group">
<a data-underline-link href="#" class="underline-link">Underline Link</a>
<a data-underline-link="alt" href="#" class="underline-link">Animation</a>
</div>
```


## Implementation

### Base attribute

Add `data-underline-link` to any link you want to animate with the default underline style.

```html
<a data-underline-link href="#" class="underline-link">Underline Link</a>
```

### Alt variation

Add `data-underline-link="alt"` to use the alternate underline transition.

```html
<a data-underline-link="alt" href="#" class="underline-link">Animation</a>
```
