1
Single-beat scenes are unchanged — the third column simply doesn't exist. No visual noise for the majority of scenes.
2
2 beats = one row of cells, 22px each. The row stays at its natural 45px height — the cells stretch to fill it.
3
3–4 beats = 2×2 grid. Odd beat counts leave the last cell empty (invisible filler). Row height grows to accommodate two rows.
4
5 beats = 3-row tall. Each row is still 2 cells wide. The row height grows further. The info column and number stay vertically centered.
5
Active beat highlighted in accent color — only the current beat cell is lit. Scene row itself shows active state if any of its beats is active.
